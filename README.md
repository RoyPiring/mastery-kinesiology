## What this domain covers

Biomechanics modeling (OpenSim, OpenCap), wearable and movement analytics, and machine learning on movement. Each build applies the field, ships a runnable artifact with a concrete self-check, and proves the skill with a teach-back.

**What it isn't.** A degree or credential. A claim past what each build's self-check proves. Not medical, legal, or financial advice.
